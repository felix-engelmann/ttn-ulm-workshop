# TTN Ulm Workshop


## Münster-Platine

- Flashen (Datenübertragung) und Laden
- Laden mit USB
- Akku via JST Stecker
- Arduino Pro Mini
- RFM95 für Lora

## Sensoren

1. BME280
2. Magnetschalter
3. Wassersensor
4. Bodenfeuchtesensor

## Arduino IDE
* Board einstellen
    * Tools -> Board -> Arduino Pro or Pro Mini
    * CPU: Atmega328, 3.3V, 8Mhz
